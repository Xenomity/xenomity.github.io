ss : 설치된 모든 번들의 id, status, name을 보여준다.  
start [id] : 번들 시작.  
stop [id] : 반들 종료.  
init : 모든 번들 제거.  
install [bundle] : 번들 설치.  
uninstall [bundle] : 번들 제거.  
diag [id] : 번들의 문제 분석.  
refresh [id] : 번들정보 갱신.  
update [id] : 번들 수정내용 갱신.  
bundle [id] : 번들 상세 내용.
